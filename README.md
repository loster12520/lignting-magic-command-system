### 这是什么？

这是一个使用纯文字指令加载封装好的插件的系统

大概使用类似下面这样子的指令，就能够调用系统里面对应的功能

```
print 聆听大笨蛋
```

这里是向系统里面输出了一段信息：“聆听大笨蛋”

（很可惜这个有趣的功能现在并没有实装，且大概以后只会以彩蛋出现）

当然，实际的应用场景并不是这样

客户端会向服务端发送一段信息，不出意外，就是上面介绍的这段指令

然后服务端就会搜索是否符合已经加载的插件所定义的指令

然后执行指令，最后向客户端发送结果
